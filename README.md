SEO-optimalizált egyoldalas (one-pager) weboldal **Rónay Barbara** dietetikusnak / Naturmed Health – Budapest.

- Célkulcsszó: **dietetikus Budapest**
- Statikus HTML/CSS, minimális JS (mobil menü)
- Teljes structured data (MedicalClinic, Person, FAQPage, OfferCatalog, WebSite, BreadcrumbList)
- `index.html` · `robots.txt` · `sitemap.xml` · `favicon.svg`
